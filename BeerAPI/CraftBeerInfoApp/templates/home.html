Craft Beer API

Welcome to our Beer API!

We have gathered data on local breweries in Gothenburg from systembolaget.se, and you can access it through our API.

This is an API of Beer data.
The beers are from local breweries in Gothenburg. Webscraped from systembolaget.se

See API actions below.

API actions

  1. Under the 'GET' heading, you can find lists of breweries and beers.
  2. If you want to add new beer or brewery objects, go to the 'POST' heading. Remember that your data needs to be in the JSON list format: [ {'key1': 'value one'}, ]{'key2': 'value two'} ]
  3. If you want to change the data in an existing beer object, go to the 'PUT' heading and choose the ID of the beer you want to update. You'll be taken to a page where you can make your changes and save them.
  4. Finally, if you want to delete an existing beer object, go to the 'DELETE' heading and choose the ID of the beer you want to delete. You'll be taken to a page where you can confirm that you want to delete the selected beer.

We hope you find our Beer API useful and easy to use!
